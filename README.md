# ITD Banner Picker

Данное приложение позволяет каждому изменить баннер своего аккаунта в ИТД на картинку в формате .png, .jpg и .gif.

![Preview](https://github.com/travkinastesha/itd-banner-picker/blob/main/preview.png?raw=true)

## Как использовать?

1. В строке ввода сверху введи свой authorization токен. Гайд по его поиску на фото:
   ![Guide](https://github.com/travkinastesha/itd-banner-picker/blob/main/guide.png?raw=true)

  > [!IMPORTANT]  
  > Authorization токены часто обновляются, поэтому перед загрузкой баннера следует проверять наличие нового токена, иначе загрузить баннер попросту не выйдет

2. Нажми на поле вставки изображения и выбери то, какое тебе по душе (поддерживаются .png, .jpg и .gif).
  > [!WARNING]  
  > В данный момент предпросмотр гифок в своём приложении недоступен в том виде, в котором следовало бы. Рендерится одна статичная картинка.

3. Готово! Осталось только нажать на кнопку "Установить баннер".
